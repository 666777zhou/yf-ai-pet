### 定位对比

| 维度 | 标准款 | 定制款 |
|------|--------|--------|
| **目标客户** | 养猫人群，想要"猫的陪伴感" | 特定一只猫的主人，想要"这只猫的延续" |
| **外观** | 品种特征准确，不追求个体还原 | 按照片逐个特征 1:1 还原 |
| **像的程度** | "这是一只英短" | "这就是我家猫" |
| **生产方式** | 工厂批量（50-100 只/款） | 羊毛毡师手工（每只独一无二） |
| **交付周期** | 现货 / 1 周内 | 3 周 - 3 个月 |
| **售价** | ¥800 - 1500 | ¥3000 - 8000 |
| **毛利** | 中等（标准化成本低） | 高（手工成本高，溢价空间大） |
| **AI 人格** | 品种通用猫人格 | 客户描述定制猫人格 |

## 2. 标准款：品种选择

### 2.1 品种清单（按中国市场热度排序）

| # | 品种 | 核心外形特征 | 毛型 | 可选配色 | 市场覆盖 |
|---|------|-------------|------|---------|---------|
| 1 | **英国短毛猫** | 圆脸胖腮、体型敦实、短毛 | 短毛 | 纯蓝 / 银渐层 / 金渐层 | 🔥 最大市场 |
| 2 | **布偶猫** | 长毛、蓝眼、重点色、体型大 | 长毛 | 海豹重点 / 蓝重点 / 手套色 | 🔥 热门品种 |
| 3 | **中华田园猫** | 短毛、匀称体型 | 短毛 | 橘猫 / 狸花 / 黑白 | 🔥 最大基数 |
| 4 | **美国短毛猫** | 银虎斑纹、体型精干 | 短毛 | 银虎斑 / 棕虎斑 | 热门 |
| 5 | **拿破仑** | 短腿、圆脸扁鼻、长毛 | 长毛 | 纯白 / 乳色 | 差异化品种 |
| 6 | **暹罗猫** | 瘦长体型、重点色、蓝眼 | 短毛 | 海豹重点 / 蓝重点 | 小众忠诚 |

> **5 个品种 × 平均 2.5 种配色 ≈ 13 个 SKU，但只需开发 5-6 套模具。**
>
> 配色变化在工厂端只需换面料颜色，不涉及重新打版。

### 2.2 品种选择逻辑

- **英短 + 布偶 + 田园**：三家覆盖中国宠物猫市场的 ~60%+
- **美短**：北美最受欢迎品种，有出口潜力
- **拿破仑**：差异化品种，竞争对手少，且是你最熟悉的品种（你家猫）
- **暹罗**：瘦长体型与其他品种形成互补，展示产品多样性

### 2.3 每个品种的开发流程

```
AI 生成品种标准 3D 模型（Meshy/Tripo，文字描述+参考图）
        │
        ▼
Blender 精修（确保品种特征准确：脸型、耳位、体型比例）
        │
        ▼
导出多角度渲染图 + 三视图尺寸图
        │
        ▼
发给 1688/Alibaba 毛绒玩具工厂
  附带需求文档：
  · 面料规格（长毛/短毛，颜色 Pantone 色号）
  · 尺寸（体长 35cm，各部位比例）
  · 腹部隐形拉链（20cm，用于装电子元件）
  · 头身之间线缆通道（Φ15mm）
  · 喉咙位置扬声器透声区（微孔网布）
  · 耳朵内部舵机安装口袋
  · 内部填充物：仅薄层棉花做缓冲，主体中空
        │
        ▼
工厂打样（7-15 天，$50-200/款）
        │
        ▼
收到样品 → 装入电子套件实测 → 反馈修改意见
        │
        ▼
1-2 轮修改后确认样品 → 小批量下单（50-100 只/款）
        │
        ▼
到货 → 装电子套件 → 品控 → 入库
```

## 3. 定制款：1 对 1 还原

### 3.1 流程

```
客户下单
  │
  ├── 提交材料：猫的多角度照片（正脸/侧脸/全身/俯视/眼部特写）
  ├── 提交猫的性格描述（粘人/高冷/话痨/胆小……）
  └── 付款（定金 50%）
        │
        ▼
你（AI 猫品牌方）的处理：
  1. 用 AI 生成猫的 3D 参考模型（方便与羊毛毡师沟通）
  2. 整理客户需求文档（猫的特征描述 + 特殊要求）
  3. 对接合作的羊毛毡仿真宠物定制师
        │
        ▼
羊毛毡师制作（3 周 - 3 个月，看排期）：
  · 制作空心羊毛毡猫壳体（腹部留拉链）
  · 头部预留眼睛/摄像头孔位
  · 身体预留线缆通道
  · 对照照片反复调整脸部特征
        │
        ▼
你收到成品 → 外观验收 → 装入电子套件 → 烧录定制猫人格
        │
        ▼
发货 → 客户验收 → 付尾款
```

### 3.2 合作羊毛毡师筛选标准

- 淘宝/闲鱼上有真实好评的仿真宠物定制师
- 能提供过往作品集（重点看猫）
- 愿意接受"空心壳体 + 拉链"这个新做法
- 沟通顺畅，能理解电子元件空间需求

### 3.3 定价建议

| 规格 | 价格 |
|------|------|
| 头部定制（含脸部精细还原） | ¥1500-2500 |
| 全身定制（体长 35cm，含头部） | ¥3000-5000 |
| 全身定制 + 特殊要求（动作/姿势定制等） | ¥5000-8000 |

> 定制款的定价逻辑：羊毛毡师费用（¥500-1500）+ 电子套件（¥200）+ 你的服务费（沟通/AI 建模/集成/利润）。3-5 倍毛利是合理的。

## 7. 路线图

```
Phase 1：本周
  ├── 快速原型：毛绒玩具 + 电子 → 链路跑通
  └── 选定第一个品种 → 开始 AI 生成 3D 模型

Phase 2：第 2-3 周
  ├── 第一个品种的 3D 模型 + 工厂打样
  ├── 电子套件标准化（PCB 打样？还是手工焊）
  └── 服务器端猫人格 v1.0

Phase 3：第 4-6 周
  ├── 第一个品种样品到货 → 装电子 → 测试
  ├── 其余 4 个品种陆续打样
  └── 联系 1-2 个羊毛毡师，验证定制款可行性

Phase 4：第 7-8 周
  ├── 标准款首批小批量生产（50 只/款，先做 1-2 个品种试水）
  ├── 定制款接受预定
  └── 上线销售（淘宝/闲鱼/小红书）
```

## 8. 关键风险与应对

| 风险 | 影响 | 应对 |
|------|------|------|
| 工厂还原度不够 | 标准款"不像" | 打好提前量，1-2 轮修改，不行换工厂 |
| 羊毛毡师不接受"空心"做法 | 定制款无法交付 | 先找 2-3 个师同时沟通，提高成功率 |
| 电子套件故障率高 | 售后成本 | 模块化设计，坏哪个换哪个 |
| 麦克风在毛绒里拾音差 | 语音交互体验差 | 快速原型阶段重点测试，调整麦克风位置和增益 |
| 舵机噪音大 | 用户体验差 | 用软胶垫隔振，或在固件中限制舵机速度降低噪音 |
